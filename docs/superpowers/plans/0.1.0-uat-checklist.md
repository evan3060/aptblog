# aptblog 0.1.0 UAT 检查清单

> 封仓前人工验收清单。自动化验证项由独立子 agent 执行，人工视觉核验项由用户在浏览器中核验。

## 1. 自动化验证（子 agent 执行）

### 1.1 HTTP 批量检查

- [ ] 中文文章页 18 篇全部 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/zh/<slug>.html`
- [ ] 英文文章页 18 篇全部 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/en/<slug>.html`
- [ ] 图片资源 40 张全部 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/images/<slug>/<name>.png`
- [ ] 首页 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/`
- [ ] About 页 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/about/`
- [ ] sitemap.xml 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/sitemap.xml`
- [ ] atom.xml 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/atom.xml`
- [ ] search.xml 200：`curl -s -o /dev/null -w "%{http_code}" https://blog.aptbot.de/search.xml`

### 1.2 Broken Link 检查

- [ ] 爬取首页所有 `<a href>` 链接，验证无 404
- [ ] 抽检 3 篇文章页内部链接，验证无 404

### 1.3 Frontmatter 断言

- [ ] 文章总数 = 36（18 zh + 18 en）
- [ ] 每篇文章 frontmatter 含 slug 映射正确（文件名与 URL 一致）
- [ ] 每篇文章 frontmatter 含 categories 字段（track + chapter）
- [ ] 每篇文章 frontmatter 含 date 字段（YYYY-MM-DD 格式）
- [ ] 每篇文章 frontmatter 含 lang 字段（zh 或 en）
- [ ] 每篇文章 frontmatter 含 permalink 字段（`<lang>/<slug>.html`）

### 1.4 构建输出检查

- [ ] `npx hexo generate` 输出无 ERROR
- [ ] `npx hexo generate` 输出无 WARN（或 WARN 可解释）
- [ ] public/ 文件总数 > 200

### 1.5 外部资源检查

- [ ] `grep -r "fonts.googleapis.com" public/` 返回 0 匹配
- [ ] cdnjs.cloudflare.com 可访问性（中国网络环境验证）

### 1.6 HTTP 重定向

- [ ] `curl -s -o /dev/null -w "%{http_code}" http://blog.aptbot.de/` 返回 301

### 1.7 SSL 证书

- [ ] `echo | openssl s_client -connect blog.aptbot.de:443 2>/dev/null | openssl x509 -noout -dates` 显示 notAfter 在未来
