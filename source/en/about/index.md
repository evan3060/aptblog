## About aptblog

aptblog is an independent tech blog extended from the [aptbot](https://github.com/anthropics/aptbot) project, focused on the distillation and sharing of **Agent practices** and **AI coding experience**.

Here you will find in-depth discussions on the following topics:

- **Agent Architecture Design**: From what an Agent is, to the design philosophy behind core modules such as the Provider system, Tool system, Memory system, Skills system, and Hook system.
- **AI Coding in Practice**: First-hand engineering experience including coding accuracy, spec document management, long-term iteration, and handling edge cases.
- **System Evolution and the Future**: Channel transport, sessions and multi-user support, security models, error handling and streaming UX, as well as the future roadmap for Agents.

This site is built with [Hexo](https://hexo.io/) static site generator and uses the [NexT](https://theme-next.js.org/) theme, striving for a clean, modern, fast-loading experience with good readability.
